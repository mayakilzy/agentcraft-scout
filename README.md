# AgentCraft Scout

**B2B Intelligence Platform** — منصة استخبارات تجارية مبنية على الذكاء الاصطناعي.

🌐 **Live Demo:** https://scout.agentcraft.info

## What it does

AgentCraft Scout operates with two independent commercial arms from a single shared infrastructure:

- **Industrial Arm** — Scrapes and analyzes Turkish industrial companies across 81 provinces for B2B export marketing
- **Web Opportunities Arm** — Discovers companies without websites or with broken sites for web design services

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS 4 + Radix UI
- FastAPI (Python) Backend
- NocoDB Database
- 8 Languages with full RTL Arabic support

## Setup

```bash
cp .env.example .env
# Add your API keys to .env
npm install
npm run dev
```
